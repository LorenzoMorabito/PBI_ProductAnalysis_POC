# PBI Marketing Domain

Area di dominio per i package marketing e per l'eventuale futura estrazione di consumer marketing dedicati.

## Contenuti

- `catalog/`
  registro dei package marketing disponibili
- `packages/`
  source of truth dei package marketing
- `projects/`
  area futura per consumer o core marketing dedicati
